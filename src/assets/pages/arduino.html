O que é arduino ?
O Arduino é uma plataforma de prototipagem open-source, seu software multiplataforma é flexível e fácil de usar. Ele é destinado para pessoas interessadas em criar objetos ou ambientes interativos. Com o Arduino, você pode interagir com luzes, motores entre outros objetos eletrônicos.
Ao lado, temos uma placa de arduino "crua", tambem com uma legenda dos componentes já contidos nele.
Nossos projetos
Projeto Arduino luz de emergência:

Criamos um sistema para arduino, que faça o sensor de luz fotovoltaico identificar a ausência e a presença de luz, após identificar a ausência, uma carga será emitida fazendo com que o LED vermelho se acenda, quando o sensor identificar presença de luz, essa carga é interrompida fazendo a luz do LED apagar. Materiais:

  1. Protoboard (1)
  2. LED de cor vermelha (1)
  3. Cabos (9)
  4. Resistor (2)
  5. Sensor de luz fotovoltaico (1)
  6. Arduino (1)
Projeto Arduino sinal de trânsito:

Criamos um sistema de loop para arduino, esse programa emite uma carga, que faz umas das três luzes de LED acenderem, após um determinado tempo, essa carga e interrompida fazendo a luz se apagara, em seguida outra carga é emitida fazendo outra das três luzes acender, e esse ciclo se repete infinitamente, até que o programa seja interrompido.

  1. (1) Protoboard
  2. (3) Leds de cores difernetes
  3. (10) Cabos
  4. (1) Placa de arduino
  5. (3) Resistores
Projeto Arduino motor bidirecional:

Criamos um sistema para arduino, que faz uma carga seja ativada ao apertar um dos dois botões, essa carga ativa o motor que faz ele girar em sentido horário, ao apertarmos o outro botão uma carga diferente é emitida, fazendo com que o motor gire no sentido anti-horaria, ao soltar qualquer um dos botões a carga e interrompida e o motor para de funcionar.

  1. (1) Ponte L293D
  2. (2) Botoes
  3. (18) Cabos
  4. (1) Motor
  5. (1) Bateria 9V
  6. (2) Resistor
  7. (1) Protoboard
  8. (1) Arduino
